# profile-insights

## What

転職活動で発生する「閲覧・反応・企業情報」を  
構造化して分析するための個人プロジェクト。

## Why

転職市場の評価はブラックボックスに近く、  
「何が刺さったのか」が本人からは見えない。  
その構造をデータとして扱いたかった。

## Current Status

- Wantedly のデータ取り込みまで実装済み
- 分析・可視化・RAG は検証フェーズ

## Design Notes

- 実運用よりも「構造の理解」を優先
- 後続で Rust / PGVector への移植を想定

## Roadmap

- Python で RAG の最小構成を検証
- ベクタ化・類似企業探索
- 仮説 → 検証 → 修正のループを回す
